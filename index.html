Overview

Our goal was to create a web platform that friends could use to create, share and form their own online radio station. In this platform, friends could use an existing social media platform, such as Facebook, to colaboratively operate this station.

Station building

Users can like or dislike songs which will in turn shape what music the station plays. Users can also leave a comment on a song if they so choose. These comments and ratings are available to see by others on the station.

User control and Regulation

Using Facebook authentication, people can keep stations private and allow only their friends and other approved users to be able to listen. This allows the users to keep the stations as intmate or public as they like.

Under the Hood

By using Spotify and Facebook API's we can lighten the load that would prohibit large amounts of people to use the system. In this way, we can support many users concurrently. Since we are using Spotify as our source of music, we need not need to worry about users trying to download or save songs as Spotify is a streaming service.
